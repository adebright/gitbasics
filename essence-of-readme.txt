Essence of Readme.md 
==================== 

Readme.md file is very important to any project. This file serves as the documentation for any project. Alongside wikis , Readme.md is aimed
at making a project to be understood by collaborators.
It is the first thing people will read when they first see your work. 
How to create readme.md 
----------------------- 
You can create a readme.md file while initializing your repository. Just select initialize with a reame.md when creating 
your repository .
More [information] on the importance of documentation can be found [here](https://guide.github.com/features/wikis)
